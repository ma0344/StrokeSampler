# Issue: 書き出したPNGを別アプリで開いて検証する

## 概要
生成したPNGを外部アプリで開き、期待どおりのサンプルになっているかを確認する。

## 目的（完了条件）
- 素材用PNGが透明背景で保存されている。
- 確認用PNGが白背景で保存されている。
- 確認用PNGにテキストラベルが含まれている。
- 解像度（ピクセルサイズ）が期待どおり。
- サンプル線が重ならず、段階差が確認できる。

## ロードマップ小項目（割り当て）
- 画面で生成して表示確認する
- ExportしてPNGを別アプリで開き、品質確認（透明背景/白背景、解像度、ラベル位置など）を行う

## 作業内容
1. アプリを起動する。
2. `生成` を押してサンプル線が描画されることを確認する。
3. `素材用PNG（透過）` を押してPNGを書き出す。
4. `確認用PNG（白+ラベル）` を押してPNGを書き出す。
5. 画像ビューア/編集ソフトで両方を開き、以下を確認する。
   - 素材用: 透明背景（チェッカーパターン等で表示される）
   - 確認用: 白背景 + テキストラベルが表示される
   - ピクセルサイズが指定値と一致する

## 影響範囲
- なし（手動検証のみ）

## メモ
- 品質確認のため、同一条件で複数回書き出して差分がないことも確認する。
