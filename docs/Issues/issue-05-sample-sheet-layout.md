# Issue: サンプルシートのレイアウトルーチンを実装する

## 概要
Pressure 4段階などのサンプルストロークを、重ならないように一定の規則で配置して生成する。

## 目的（完了条件）
- Pressure `0.2 / 0.5 / 0.8 / 1.0` のサンプルが縦方向に並ぶ。
- 生成結果が毎回同じ配置になる（再現性）。
- PNG書き出し時に“同じ並び”で描画できる（保持/取得の方針が定まっている）。

## ロードマップ小項目（割り当て）
- 後でPNGに“同じ並び”で描画できるよう、生成した `InkStroke` 群を取得/保持する方針を決める（例：`InkPresenter.StrokeContainer.GetStrokes()` を都度読む）

## 作業内容
- ループで Pressure を変化させつつ、Y座標をずらしてストロークを生成する。
- 線長・間隔・余白を定数化する。

## 影響範囲
- `MainPage.xaml.cs`

## メモ
- 後でラベルを焼き込む場合、レイアウト情報（y位置など）を再利用できる形が望ましい。
